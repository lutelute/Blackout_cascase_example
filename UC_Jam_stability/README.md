# IEEE 30-Bus 統合電力系統解析ツール

## 概要

IEEE 30-Bus標準系統をベースとした、電力系統の包括的な解析・可視化ツールです。発電計画から事故時の安定度解析まで、電力系統運用の主要な側面をインタラクティブに学習・検証できます。

## 主要機能

### 1. UC計画（Unit Commitment）
発電機の起動停止計画と経済負荷配分を最適化します。

- **24時間スケジュール管理**: 各時間帯の発電機ON/OFF状態と出力を視覚化
- **経済負荷配分**: 限界費用（λ）ベースの最適配分アルゴリズム
- **最適化手法**: 
  - 優先順位法（Merit Order）
  - 動的計画法（DP）
  - 区分線形MILP
- **負荷パターン**: 典型的な日負荷曲線、産業用、住宅用、ダックカーブ
- **コスト計算**: 燃料費（2次関数）、起動費、停止費

### 2. 系統混雑解析
送電線の潮流と容量制約を解析します。

- **DC潮流計算**: Bマトリクスによる高速近似計算
- **系統図可視化**: 
  - 潮流方向の矢印表示
  - 負荷率による線の太さ・色分け
  - バス電圧による色分け
- **カスケード停電シミュレーション**: 過負荷による連鎖的な線路脱落
- **送電損失計算**

### 3. 電圧安定度解析
系統の電圧安定性を評価します。

- **電圧プロファイル**: 全30バスの電圧分布
- **dQ/dV指標**: 電圧安定度マージンの定量評価
- **PV曲線**: 負荷増加に対する電圧崩壊特性
- **QV曲線**: 無効電力-電圧特性（ノーズカーブ）
- **最脆弱バスの特定**: 電圧崩壊の起点となるバス

### 4. 事故時安定度解析
発電機の過渡安定度を評価します。

- **P-δ曲線**: 等面積法による安定度解析
- **CCT計算**: 限界遮断時間（Critical Clearing Time）
- **N-1解析**: 単一設備脱落時の系統安定性評価
- **時間領域シミュレーション**: 位相角の動的応答

## IEEE 30-Bus系統について

### 系統構成
- **バス数**: 30
- **ブランチ数**: 41（送電線＋変圧器）
- **発電機数**: 6（Bus 1, 2, 5, 8, 11, 13）
- **総負荷**: 約283.4 MW（基準状態）
- **基準容量**: 100 MVA

## 技術詳細

### 潮流計算（DC近似）
```
P = B * θ
```
- Bマトリクス: サセプタンス行列
- θ: 位相角ベクトル
- ガウス・ザイデル法による反復計算

### CCT計算（等面積法）
```
CCT = √(2H * (δcr - δ0) / (ω0 * Pm))
```
- H: 慣性定数 [s]
- δ0: 初期位相角
- δcr: 臨界位相角 (π - δ0)
- Pm: 機械入力 [pu]

### 電圧安定度指標
```
dQ/dV = 2V * ΣB - Qload/V
```
dQ/dV > 0 で電圧安定、0に近づくほど不安定

## 使用方法

1. **ブラウザで開く**: HTMLファイルをダブルクリック
2. **タブ切替**: 上部のタブで解析モードを選択
3. **パラメータ調整**: スライダーや入力欄で条件を変更
4. **解析実行**: 各種ボタンで計算・シミュレーション実行

### UC計画タブ
- 時刻スライダーで24時間を移動
- 「再生」ボタンで自動進行
- スケジュールグリッドをクリックでON/OFF切替
- 「最適化実行」で自動スケジューリング

### 系統混雑タブ
- 「ブランチ脱落」で選択した線路を切り離し
- 「カスケード開始」で連鎖停電シミュレーション
- 負荷倍率を変更して過負荷状態を作成

### 電圧安定度タブ
- 「PV曲線計算」で電圧崩壊点を探索
- 「QV曲線計算」で無効電力余裕を確認
- 負荷増加スライダーで電圧低下を観察

### 事故時安定度タブ
- 「N-1解析」で全線路脱落ケースを評価
- 「事故シミュレーション」で時間応答を確認
- 事故継続時間を変えてCCTとの比較
